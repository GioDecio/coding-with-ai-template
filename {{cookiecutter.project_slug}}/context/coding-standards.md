# Coding Standards

This project maintains coding standards for different technology stacks. Choose the relevant guide based on the technology you're working with:

## Stack-Specific Standards

- **[Python](./coding-standards-python.md)** - Python 3.11+, type hints, UV package manager
- **[TypeScript/React/Next.js](./coding-standards-typescript.md)** - Server-first Next.js, Tailwind CSS v4, Prisma ORM

## Universal Principles

- Keep it simple and readable
- Use consistent naming conventions
- Always add type hints/type safety
- Test your code
- No commented-out code
- Keep functions focused and under 50 lines when possible
- No unused imports or variables
